Dodge the Creeps!

Beginner game development with Godot

Details

This project was an assignment for my game development class in my master's program. In it we were challenged with following this tutorial to develop our first 2D game. After we completed the tutorial we had to make additional adjustments to the game like adding new timers, enemy types, and collectible items. I found this to be a fascinating introduction to game development, and felt like the project provided me with a nice mixture of guidance and self teaching opportunities. by the end of the project I felt like I'd had that "Aha!" moment where everything started to click together. This was our first class project, and I'm eagerly looking forward to building on the knowledge I gained here when we make more games in the future.

What I learned

  • The basics of GDscript, Godot's beginner dev programming language
  • How to properly build scenes and connect them together with Godot's heirarchy system
  • Core functionalities of the Godot development environment (animation, binding assets to nodes, etc.)

Tools Used

  • Godot Development Environment GUI (for creating and binding nodes, and managing assets)
  • GD Script (for coding game behavior)
  • Clip Studio Paint (for asset editing and creation)
